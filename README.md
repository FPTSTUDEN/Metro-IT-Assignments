# How to work on this repo

## Setup

- Go to ```https://wokwi.com/projects/354682548125570049```
- On the web, make new file ```encappedhw.py``` and paste content from ```encappedhw.py```

## Working

- Firstly, read ```main_example.py``` for example commands
- On this repo, edit on ```main.py``` of the exercise that you want (for example, ex1.1 -> edit ```main_1.1.py```)
- To test, paste content from there to the ```main.py``` file on wokwi and run
